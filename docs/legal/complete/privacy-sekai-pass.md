## 我们收集的信息

### 您主动提供的信息

为了提供用户认证服务，我们需要收集以下信息：

| 信息类型 | 用途 | 说明 |
|---------|------|------|
| **用户名** | 唯一标识用户 | 必填项，用于登录和识别身份 |
| **电子邮箱** | 账户唯一标识、找回密码 | 必填项，用于账户验证和通知 |
| **密码** | 验证用户身份 | **不存储明文密码**，仅存储经过 PBKDF2 算法哈希处理后的凭据 |
| **显示名称** | 个性化展示 | 可选项，用于在界面中显示用户昵称 |
| **头像URL** | 个性化展示 | 可选项，用于显示用户头像 |
| **授权信息** | OAuth 2.0 流程 | 记录您授权给第三方应用的信息 |

### 自动收集的信息

当您使用本服务时，系统会自动收集以下信息：
- **网络日志**：IP 地址、User-Agent、访问时间等（由 Cloudflare 基础设施自动记录）
- **会话 Cookie**：用于维持您的登录状态（Session ID）
- **安全验证数据**：Cloudflare Turnstile 验证令牌（用于区分人类用户与机器人）

### 我们不主动收集的信息

**本服务不会主动收集以下信息：**
- 真实姓名、身份证号、电话号码
- 精确地理位置数据
- 支付和财务信息
- 生物识别信息

## 信息的存储与安全

### 存储位置与跨境传输

**您的账户数据存储在 Cloudflare D1 数据库中。** 根据 Cloudflare 的服务架构，您的数据可能存储在以下地区的服务器：
- 美国
- 欧盟成员国
- 亚太地区（包括新加坡、日本等）

**跨境传输说明**：
- **数据接收方**：Cloudflare, Inc.（美国公司）
- **传输机制**：我们依据 Cloudflare 的标准合同条款（Standard Contractual Clauses）进行数据跨境传输
- **安全保障**：Cloudflare 承诺提供与中国法律相当的个人信息保护水平，并遵守 GDPR 等国际数据保护标准
- **您的权利**：您有权了解接收方的名称、联系方式、处理目的、处理方式、个人信息的种类以及您向接收方行使个人信息权利的方式和程序等事项
- **联系方式**：如需了解更多跨境传输信息，请通过本政策的联系方式与我们联系

**重要提示**：本服务为个人开发的非商业项目，未进行国家网信部门的安全评估。如您对数据跨境传输有顾虑，请谨慎使用本服务。

### 数据安全

**密码安全**：我们采用行业通用的 PBKDF2 算法对您的密码进行单向哈希加密存储。**管理员无法查看您的明文密码。**

**传输安全**：所有数据传输均经过 HTTPS 加密协议。

**会话安全**：登录会话 Cookie 设置了 `HttpOnly` 和 `Secure` 属性（在生产环境中）。

### 数据保留

- **活跃账户**: 只要您维持账户，我们就会保留您的账户信息
- **注销账户**:
  - 您可通过联系方式申请注销账户
  - 注销后，您的个人信息（用户名、邮箱、密码哈希、显示名称、头像URL）将在 **30天内** 从生产数据库中永久删除
  - 备份系统中的数据将在 **90天内** 自动清除
  - 法律要求保留的日志数据（如安全审计日志）将保留 **6个月**，到期后自动删除

### 年龄限制

**如果您未满 14 周岁，请勿使用本服务。** 如果您是 14 周岁以上、18 周岁以下的未成年人，请在法定监护人的陪同下阅读本隐私政策，并在取得监护人书面同意后使用本服务。

本服务依赖用户自我声明年龄。我们不主动验证用户年龄，但如发现未成年人在未取得监护人同意的情况下使用本服务，我们将立即采取措施保护其权益。

### 特殊保护

我们不会主动收集未成年人的个人信息。如发现在未取得监护人同意的情况下收集了 14 周岁以下未成年人的信息，我们将立即删除相关数据。
