# 本地存储补充条款

> 适用于：25時作業風景 (25ji.nightcord.de5.net)

---

## 我们收集的信息

### 本地存储的信息

为了提供更好的用户体验，本服务使用浏览器的本地存储功能（`localStorage` 和 `sessionStorage`）保存您的使用偏好：

| 信息类型 | 存储方式 | 存储目的 | 存储期限 |
|---------|---------|---------|---------|
| 音量设置 | localStorage | 记住您的音量偏好 | 持久存储，直至手动清除 |
| 播放模式 | localStorage | 记住顺序/随机/循环模式 | 持久存储，直至手动清除 |
| 上次播放的歌曲 | localStorage | 恢复播放进度 | 持久存储，直至手动清除 |
| 时区偏好 | localStorage | 显示偏好的时区 | 持久存储，直至手动清除 |
| 番茄钟设置 | localStorage | 自定义工作/休息时长 | 持久存储，直至手动清除 |
| 番茄钟状态 | sessionStorage | 页面刷新后恢复计时 | 会话结束时清除 |
| 世界时钟时区选择 | localStorage | 记住选择的时区 | 持久存储，直至手动清除 |
| 用户昵称 | localStorage | 在实时状态中显示您的名称 | 持久存储，直至手动清除 |
| 成就数据 | localStorage | 记录已解锁的成就 | 持久存储，直至手动清除 |
| 学习统计 | localStorage | 番茄钟累计、学习时长、连续天数等 | 持久存储，直至手动清除 |
| 活动记录 | localStorage | 记录最近的学习活动 | 持久存储，直至手动清除 |

### 我们不主动收集的信息

**本服务不会主动收集以下信息：**
- 个人身份信息（姓名、身份证号、邮箱、电话等）
- 精确地理位置数据
- 支付和财务信息
- 社交媒体账户信息
- 设备唯一标识符
- 生物识别信息

## 信息的存储与安全

### 存储位置

**所有用户偏好数据均存储在您的本地设备上，不会传输到我们或任何第三方的服务器。**

- **localStorage**：持久存储，除非您手动清除，否则会一直保留
- **sessionStorage**：会话存储，关闭浏览器标签页后自动清除

### 数据安全

由于数据仅存储在您的本地设备上，数据安全性取决于您设备的安全状况。

我们建议您：
- 使用安全的网络环境访问本服务
- 定期更新浏览器以获取最新的安全补丁
- 避免在公共设备上保存敏感偏好设置

### 数据传输

**本服务不会将您的本地存储数据传输到任何远程服务器。** 但请注意，您在使用本服务时，网络请求（如加载页面、获取资源）可能会经过第三方服务（如 Cloudflare CDN）。

## 您的权利

### 访问您的数据

您可以通过浏览器的开发者工具查看存储在 localStorage 和 sessionStorage 中的所有数据：

1. 在浏览器中按 `F12` 打开开发者工具
2. 转到 `Application`（应用程序）或 `Storage`（存储）标签页
3. 在左侧菜单中选择 `Local Storage` 或 `Session Storage`

### 删除您的数据

您可以随时删除本服务存储的数据：

**方法一：通过浏览器设置**
- Chrome: 设置 → 隐私设置和安全性 → 清除浏览数据
- Firefox: 设置 → 隐私和安全 → 清除数据
- Safari: 偏好设置 → 隐私 → 管理网站数据
- Edge: 设置 → 隐私、搜索和服务 → 清除浏览数据

**方法二：通过开发者工具**
1. 打开浏览器开发者工具 (F12)
2. 转到 Application/Storage 标签页
3. 右键点击相关条目并选择删除

### 选择退出

您可以随时停止使用本服务，这将阻止任何进一步的数据存储。

如果您是欧盟用户，您享有以下权利：
- **访问权**：了解我们收集了哪些关于您的数据
- **更正权**：更正不准确的个人数据

由于本服务不收集个人身份信息，上述权利主要适用于您本地存储的偏好数据，您可以自行管理。
